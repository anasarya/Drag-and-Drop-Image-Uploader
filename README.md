# Drag-and-Drop-Image-Uploader
This project is a modern image uploader web app built using HTML, CSS, and JavaScript. It allows users to drag &amp; drop or click to upload image files with real-time upload progress simulation, image preview, and localStorage persistence. The UI uses a glassmorphism transparent background with smooth animations and an eye-catching design.

**Features**

✅ Drag & Drop or click to upload images

✅ Live preview of uploaded images (side gallery view)

✅ Animated upload progress bar

✅ LocalStorage support to keep images after page refresh

✅ Clear All button to remove saved images

✅ Fully responsive and beautifully designed with a transparent glass effect

Technologies Used

HTML5 for structure

CSS3 (Glassmorphism + animations) for styling

JavaScript (ES6) for logic and FileReader API

** Additional Info**

You can use this uploader in any web project to handle image uploads visually and interactively.
It’s lightweight, easy to customize, and built without any external libraries.
